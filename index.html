如何使用termux安装kali

首先先需要下载一个termux

直接在浏览器搜索一下就有了

然后打开我们的termux输入下边这一串命令切换清华源

sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/apt/termux-main stable main@' $PREFIX/etc/apt/sources.list apt update && apt upgrade

当然如果你安装的是zerotermux的话直接左滑找到切换源换清华源就可以了其他源可以自己换

之后就输入pkg install wget来安装wget

安装好之后就输入wget -O install-nethunter-termux https://offs.ec/2MceZWr

好了之后打出这两个命令

chmod +x install-nethunter-termux

./install-nethunter-termux

接下来就是一个很漫长的过程本人建议在晚上的时候安装

安装解压好之后输入nh -r

就能进入kali系统了
